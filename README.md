# Green Software Foundation (green-software-foundation)
The Green Software Foundation is a Linux Foundation project founded by Accenture, GitHub, Microsoft, and ThoughtWorks to build standards, tools, and best practices for reducing the carbon emissions of software. It developed the Software Carbon Intensity specification and provides patterns for building green software.
